# Desafio

Ler um número inteiro N e calcular todos os seus divisores.

## Entrada

O arquivo de entrada contém um valor inteiro.

## Saída

Escreva todos os divisores positivos de N, um valor por linha.

 

| Exemplo de Entrada | Exemplo de Saída       |
| ------------------ | ---------------------- |
| 6                  | 1<br />2<br />3<br />6 |
